## Installation

### Requirement

- php 8.0 or Higher

### Running

```sh
$ git clone https://github.com/pacoorozco/advent-of-code-php.git

$ composer install
```

```sh
$ bin/console puzzle:exec [-y|--year YEAR] [-d|--day DAY] [-p|--puzzle PUZZLE]
```

### Testing

```sh
$ vendor/bin/phpunit --testdox tests/<YEAR>/<DAY>
```
